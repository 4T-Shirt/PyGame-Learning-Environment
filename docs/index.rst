Welcome to PyGame Learning Environment (PLE)
=======================================================

**PLE** is a learning environment, mimicking the Arcade Learning Environment interface, allowing a quick start to Reinforcement Learning with Python. It focuses on allowing the practitioner to focus on the model and fast iteration cycles.

User Guide
------------

The PLE user guide below explains the different components inside of the library. It covers how to train a reinforcement learning agent, the environments structure and function, and how to write additional games for usage in the library.
